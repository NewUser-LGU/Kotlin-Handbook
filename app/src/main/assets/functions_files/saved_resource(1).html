                                      Пример из android
                                      Без наименований:

                                      val cursor = database.query(false, "table1", arrayOf("column1","column2"), "column1 =?", arrayOf("value1"), null, null, null, "10")

                                      С ними:

                                      val cursor = database.query(distinct = false, table = "table1", columns = arrayOf("column1","column2"), selection = "column1 =?", selectionArgs = arrayOf("value1"), groupBy = null, having = null, orderBy = null, limit = "10")

                                      P.s. да нормальные IDE могут сами такие аргументы подписывать, но не всегда код читается из IDE ))
