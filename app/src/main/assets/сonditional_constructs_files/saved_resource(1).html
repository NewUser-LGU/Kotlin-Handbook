            Еще when можно использовать без выражение. Это чем то похожа на if...else.

            var a = 5
            var b = 2

            when {
            (a > b) -> println("BLACK")
            (a == b) -> println("RED")
            (a < b) -> println("YELLOW")
            else -> println("null")
            }

            Console:
            BLACK
